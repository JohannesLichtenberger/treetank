TreeTank Java Bean Binding

TreeTank can read and write Java POJOs that comply with the following conventions:

  • The java bean has the usual private fields accessible through getters and setters.
  • The fields are of String, int, long, float, double, or boolean.
  • There is a default constructor with no arguments. It is implicit if no constructor is implemented and must be made explicit as soon as there are other constructor accepting arguments.
